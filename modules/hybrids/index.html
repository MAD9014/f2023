# Hybrid Overview

MAD9014 is a Hybrid course. That means that each week there is meant to be an additional hour of instructional activity with an accompanying hour of exercise work.

There are two sections to the Hybrid portion of the course.

During the first half of the semester (before the reading week), there are a series of six Core JavaScript practice exercises covering a range of basic JavaScript skills. You will need to complete and submit each of these six exercises separately.

All six must be completed and submitted by week 7. However, you can submit them as soon as you complete each one.

We strongly urge you to complete them as soon as you can. The sooner you complete these, the better equipped you will be to do the class assignments.

During the second half of the semester, there is a video to watch about the Fetch API. This video is a little over two hours long. You can watch it all at once or in parts.

After watching the video there is a single Hybrid exercise for you to complete. The exercise uses the Fetch API. Use the Fetch API video as your reference for completing the exercise.
